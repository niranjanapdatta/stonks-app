NSEpy Python Library: NSEpy is a library to extract historical and realtime data from NSE’s website. This Library aims to keep the API very simple. Close price and close date data is fetched using this library.

Analysis:
  • A python module performs the following analysis on the data (for each and every stock that exists in the database) that is fetched from NSEpy library: Volatility, Beta, Price Up and Down (Days), Average Price Up and Down (₹).

  • Another Python module performs the previously mentioned analysis, but on a single stock when requested by the client.

  • Trading date Python module processes a list of dates and returns the exact trading date required to perform analysis for six months, one year, three years, and five years.
